求最多有多少个点在同一条直线上

题目

Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.

Example

思路

计算多少个点在同一条直线上，可以转化为，一个向量上能经过多少个点，将向量移到坐标原点，问题又变成了多少个点有相同的斜率。所以现在思路就很明显了，在一个循环中分别将每个点作为坐标原点，然后在循环体内计算其他点与他的斜率，用动态规划来更新最大的点数:
$$
OPT[i] = MAX(points[i], OPT[i - 1])
$$

这里需要注意的是分母为零的情况，我的算法中处理为一个很小的小数（这里可能不够优雅，有非常小的概率会导致算法出错，可以考虑单独计数。但是我认为就算真的有这么一个点，那他和Y轴也是非常接近的，一般工程上也认为它等同于Y轴了）

这里有同学可能会想到GCD的方法，事实上对每两个点求GCD是有重复的信息计算的。而直接作差（即假定坐标原点）就可以免去这些额外的计算。

最后结果是：runtime 8 ms 95.15%, memory: 9.6 MB, 71.43%
